# Sudoku Solver
A solver of Sudoku puzzles using human techniques

*This is still a work in progress as a lot of techniques has not yet been implemented.*

## Current techniques supported
* Full house
* Naked singles
* Hidden singles
* Naked subsets
* Hidden subsets
* Candidates
